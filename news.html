How to add a post

  1. assets/images ফোল্ডারে ছবি রাখুন (যেমন: assets/images/my_post.jpg)।
  2. news.html-এ একটি নতুন
    ব্লক কপি করে শিরোনাম, তারিখ, বডি, এবং ইমেজ পাথ আপডেট করুন।
  3. git add . → git commit → git push করুন; ডিপ্লয় হলে সবার জন্য দেখা যাবে।
